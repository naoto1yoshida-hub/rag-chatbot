RAGチャットボット 完成報告
概要
手元のPDFファイルをアップロードして、その内容について質問できるチャットボットです。 複数のPDFファイルを一度に読み込ませることが可能です。

主な機能
複数PDFアップロード: サイドバーから複数のPDFファイルを一度にアップロードできます。
RAG（検索拡張生成）: アップロードされたドキュメントの内容を検索し、回答を生成します。
チャット履歴: 会話の文脈を保持しながらやり取りができます。
履歴クリア: 「クリア」ボタンで会話と読み込んだドキュメントをリセットできます。
使い方
アプリケーションを起動します。
bash
streamlit run app.py
ブラウザで http://localhost:8501 にアクセスします。
サイドバーの「API Key」欄にOpenAIのAPIキーを入力します（環境変数に設定済みの場合は不要）。
「PDFをアップロード」ボタンから、分析したいPDFファイル（複数可）を選択します。
「X個のファイルを読み込みました！」と表示されたら準備完了です。
チャット欄に質問を入力して対話を開始します。

技術スタック
Frontend: Streamlit
LLM/Embeddings: OpenAI (GPT-4o-mini)
Framework: LangChain
Vector Store: FAISS
